# mon-captcha
php验证码库

- 支持图像拖拽、数字、英文字符、数学计算、中文等多种图片验证码类型
- 支持自定义验证码存储驱动
- 支持自定义扩展验证码类型

#### 安装

```bash
composer require mongdch/mon-captcha
```
